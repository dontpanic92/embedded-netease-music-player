# embedded-netease-music-player
嵌入式网易云音乐播放器 - Embed a netease music player into your web page

![screenshot](screenshot.png)

复制粘贴下面的 HTML 代码放入页面中，将参数 `536622304` 改为需要播放的音乐 ID 即可。
